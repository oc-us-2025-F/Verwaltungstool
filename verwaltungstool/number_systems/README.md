Tool zum Üben der Umrechnung zwischen Zahlensystemen. Es setzt voraus, dass der Anwender die Rechnung anhand der zufällig generierten Zahl selbst durchführen kann.

Zweck: Das Tool dient ausschließlich der Übung. Wenn eine Umrechnung mehr als dreimal hintereinander nicht gelingt, wird empfohlen, eine Lehrkraft oder einen Dozenten zu kontaktieren, damit der Rechenweg auf Fehler überprüft werden kann.
